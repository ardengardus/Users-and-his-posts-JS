
# Пользователи и их посты

Показательное React-приложение для просмотра постов пользователей с JSONPlaceholder.

## Функционал

- Выбор пользователя из списка
- Просмотр постов выбранного пользователя
- Разворачивание полного текста поста ("Читать далее")
- Тёмный дизайн (черно-синие тона)
- Адаптивная вёрстка

## Технологии

- **React 18** — библиотека для UI
- **Vite** — сборщик проекта
- **CSS3** — стилизация и анимации
- **JSONPlaceholder API** — тестовое API

### Требования
- Node.js (версия 16 или выше)
- npm (устанавливается вместе с Node.js)

### Установка и запуск

 1. Клонируйте ссылку на репозиторий

 2. Перейдите в папку проекта
    *cd posts-app*

 3. Установите все нужные зависимости
    *npm install*
 
 4. Запустите проект
    *npm run dev*
